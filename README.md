# 🌍 Earth's Breathe — Schumann Resonance Generator

A web-based auditory frequency generator built around the **Schumann Resonances** — the electromagnetic frequencies produced by Earth's atmosphere. Inspired by [subtle.energy/earth-pulse](https://subtle.energy/earth-pulse/).

## Features

- **5 Schumann Harmonics** — 7.83 · 14.3 · 20.8 · 27.3 · 33.8 Hz
- **Binaural Beats** mode — dual-channel offset tones (best with headphones)
- **Isochronic Tones** mode — amplitude-modulated carrier (works with speakers)
- Adjustable **carrier frequency** (80–500 Hz)
- Animated **sacred geometry mandala** synchronized to active frequencies
- Live **waveform visualizer**
- Session **timer** with auto-stop
- Fully responsive — works on desktop and mobile

## Usage

Open `index.html` in any modern browser. No build step, no dependencies — single file.

1. Select one or more Schumann frequencies
2. Choose Binaural or Isochronic mode
3. Press **▶** to start
4. Adjust volume and carrier frequency to taste

> For binaural beats, use stereo headphones for the full effect.
